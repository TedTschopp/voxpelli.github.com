The above menu creates a dropdown automatically when the browser window is too small to fit all items.

Uses :first-line together with a really big word-spacing value to make all menu items that doesn't fit on one row have their own rows.

Browser Support

Only seems to support Firefox. WebKit-based browsers line breaks everything and Opera doesn't seem to like it much at all.
